## 📚 Overview

**CSSPM** (Computer Science Software Project Management) is an AI-powered chatbot designed to assist B.E. Computer Science and B.Tech Information Technology students with their Software Project Management queries.

## 🎯 Target Audience

- B.E. Computer Science Students
- B.Tech Information Technology Students  
- IT Job Seekers
- Software Project Management Professionals

## 🚀 Features

- 💬 Interactive Q&A interface powered by Qwen Chatbot
- 💡 Smart question suggestions (10 random prompts)
- 📖 Comprehensive coverage of 50+ SPM topics
- 🎨 User-friendly interface with custom styling
- 🔄 Reset and refresh functionality
- 📱 Responsive design

## 📋 Topics Covered

1. Scope of software project management
2. Project planning, monitoring and control
3. Stakeholder management
4. Risk management
5. Success criteria and failure causes
6. Software development lifecycle
7. Requirements analysis
8. Quality management
9. Agile and traditional methodologies
10. And 40+ more topics...
